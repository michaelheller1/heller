 # Simple Dashboard (Dash)

 פרויקט קטן בפייתון המשתמש ב־[Dash](https://dash.plotly.com/) כדי להציג דשבורד פשוט בדפדפן.

 ## תיאור הפרויקט

 האפליקציה מציגה:

 - **כותרת ראשית**: "Simple Dashboard"  
 - **טקסט ברוך הבא**: "Welcome to your dashboard!"  
 - **כפתור פעולה**: כפתור "Add" (בשלב זה ללא לוגיקה מאחוריו)

 הקוד המרכזי נמצא בקובץ `dashboard.py`.

 ## טכנולוגיות וכלים

 - **Python**
 - **Dash** (כולל `dash.html`)

 ## התקנה והפעלה

 1. **שכפול הריפוזיטורי**

 ```bash
 git clone https://github.com/USERNAME/REPO_NAME.git
 cd REPO_NAME
 ```

 2. **יצירת סביבה וירטואלית (אופציונלי אך מומלץ)**

 ```bash
 python -m venv venv
 venv\Scripts\activate
 ```

 3. **התקנת חבילות נדרשות**

 ```bash
 pip install dash
 ```

 4. **הרצת האפליקציה**

 ```bash
 python dashboard.py
 ```

 לאחר ההרצה, פתח דפדפן וגש לכתובת שתופיע במסוף, למשל:

 ```text
 http://127.0.0.1:8050/
 ```

 ## מבנה הפרויקט

 ```text
 .
 ├── dashboard.py   # אפליקציית Dash הראשית
 └── test.py        # סקריפט בדיקה פשוט ("hello world")
 ```

 ## התאמה אישית ורעיונות להמשך

 - **הוספת לוגיקה לכפתור "Add"** (לדוגמה, הוספת שורות לטבלה או עדכון טקסט).
 - **הרחבת הדשבורד** עם גרפים, טבלאות וקומפוננטים נוספים של Dash.
 - **הפרדת קוד** לקבצים/מודולים נוספים עם מבנה תיקיות מסודר.
